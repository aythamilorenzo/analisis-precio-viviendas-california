Descripcin de las variables

  1. longitud : Una medida de qu tan al oeste est una casa; un valor ms alto es ms al oeste.

  2. latitud : Una medida de qu tan al norte est una casa; un valor ms alto es ms al norte.

  3. housingMedianAge : Edad media de una casa dentro de una manzana; un nmero ms bajo es un edificio ms nuevo.

  4. totalRooms : Nmero total de habitaciones dentro de una manzana.

  5. totalBedrooms : Nmero total de dormitorios dentro de una manzana.

  6. Population : Nmero total de personas que residen dentro de una manzana.

  7. Households : Nmero total de hogares, un grupo de personas que residen dentro de una unidad de vivienda, para una manzana.

  8. medianIncome : Ingreso medio de los hogares dentro de una manzana de casas (medido en decenas de miles de dlares estadounidenses).

  9. medianHouseValue : Valor medio de la casa para los hogares dentro de una manzana (medido en dlares estadounidenses).

  10. oceanProximity : Ubicacin de la casa con respecto al ocano/mar.

Modelo utilizando una distribucin t de student

Probaremos a utilizar esta distribucin ya que admite ms peso en sus colas y quizs soporte de mejor manera los outliers para llegar a predicciones ms precisas.

reg.mult.tstudent <- brm(
  formula = log(median_house_value) ~ longitude + housing_median_age +
            log(total_rooms) + log(population) + median_income + ocean_proximity,
  data = housing,  
  family = student(),
  prior = c(
    prior(normal(0, 5), class = "b"),
    prior(normal(0, 5), class = "Intercept"),
    prior(student_t(3, 0, 10), class = "sigma"),
    prior(gamma(2, 0.1), class = "nu") # grados de libertad
  ),
  chains = 4,
  iter = 6000,
  cores = 4, # Ajustar segn la CPU 
  warmup = 1000,
  seed = 123
)

Conclusiones

Tras comparar ambos modelos podemos ver que a pesar de usar una t de student los outliers siguen siendo un problema como podemos ver en el diagnstico grfico, sin embargo, las cadenas para los parmetros del modelo convergen bien. Podemos afirmar que no sera mala idea utilizar este modelo ya que los grados de libertad de la distribucin son aproximadamente 7 y no es un nmero muy grande por lo que se aleja de una normal y quizs soporta de mejor manera los outliers que presentan los datos a pesar de que no lo hace de manera perfecta.

Esto puede deberse a una relacin no lineal entre las distintas variables, multimodalidad en los datos o el hecho de que pueden haber variables omitidas que son relevantes y que no estn en el modelo

A pesar de eso, creo que ambos modelos ofrecen una alternativa razonable para estimar la relacin entre el precio de las casas y las variables explicativas.
